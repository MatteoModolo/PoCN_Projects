scaling3dfitbeta.R: write an R script to simulate achlioptas processes with a Bohman-Frieze, Product Rule and Erdos-Renyi model (previously explained) of different sizes (then adjusted)
	with an union-find simulation. Take the average values of multiple realizations per model/size. By variating gamma from 0 to 0.6, and with diffenrent values of A (then adjusted)
	fit the difference of time steps between t0 as the time in which the size of the largest connected component is smaller than n^gamma and t1 as the time at which the size of the largest 
	connected component is of size C>An with a fitting with an hypothesis n^beta of the scaling  (here 2 approached has been tried, taking as beta value the value of logDelta/log n or for each
	value of n taking multiple values of n that averaged give the mean value of n and perform a maximum likelihood fitting with those realizations. Although the second apprach could be more rigorous it did'nt really brought
	significant improvements).
	 Plot then the results in a different plot for each value of A in a 3d plot showing beta as a function of n and gamma. With the obtained values fit with a 
	maximum likelihood criteria for each value of A with a model of gamma+lambda*beta=mu
PRpoly.R: write an R script to simulate achlioptas processes with a Bohman-Frieze, Product Rule and Erdos-Renyi model (previously explained) of different sizes (then adjusted)
	with an union-find simulation. Take the average values of multiple realizations per model/size. After extracting the averaged values of the largest connected component as a function of the size of the network and the value 
	of r (t/n) fit in logscale with a maximum likelihood approach to find the polynomial scaling of the size of the largest connected component as a function of the size of the network n. Compute 2 different fitting, one taking 
	fitting one polynomial model on all the data and one subdividing the data for r smaller then 0.5 and larger of 0.5 and performing 2 seperate fitting for ranges with a maximum likelihood appraoch. Then plot the results with a 
	different plot per value of r and compare the 2 models with a loglikelihood ratio. (Since unlike the BF and ER model there are no theoretical proofs of the model before and after the phase transition ofg the product rule
	model, just fitting with a powerlaw/polynomial model and verify the difference in the scaling was sufficient to capture a phase transition)
deltaScPRMax.R:write an R script to simulate achlioptas processes with a Bohman-Frieze, Product Rule, Sum Rule, Erdos-Renyi model and the Product Rule model that instead aims to speed up the emergence of the largest 
connected component (PRmax) (all previously explained) of different sizes with an union-find simulation.
Take the average values of multiple realizations per model/size. Instead of stopping as soon as delta is found, record the size of the largest connected component at regular step intervals to capture the full macroscopic trajectory.
Calculate the difference of time steps between t0 as the time in which the size of the largest connected component is smaller than n^1/2 and t1 as the time at which the size of the largest connected component is of size C>0.5n. 
Perform a maximum likelihood fitting in logscale with an hypothesis of the scaling of n^beta. Strictly restrict this linear regression fitting only to network sizes N >= 100,000 to ensure the slope is extracted from the
 asymptotic regime. For each fitting report the 95% credible interval of the found values. the plot the results

asympt.R: write an R script to simulate Achlioptas processes using the BF, PR, ER models of progressively larger network sizes using an Union-Find simulation.
For each model, network size, and step, record the size of the largest connected component across multiple independent realizations to construct an averaged trajectory.
From each averaged master trajectory, extract the phase transition boundaries in terms of edge densities with t0 as  defined as the density where the largest connected component first equals or exceeds n^1/2and t1
defined as the density where the largest connected component exceeds macroscopic size 0.5n.
Using these extracted boundary points across system sizes, perform a Finite-Size Scaling (FSS) analysis with a Maximum Likelihood linear regression approach.
 Regress the boundary edge densities against n^-1/3 to dynamically extract the asymptotic limit as for n going to infinity . For each fitted asymptotic limit, 
compute and report the 95% confidence interval. Plot then the results
